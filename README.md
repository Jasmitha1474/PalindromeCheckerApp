# PalindromeCheckerApp

// running procedure

Clone the repository
Switch to the required branch (main / welcome-page)
Run the application
Enter a word or sentence to check
View the palindrome result on the screen
// flow of project

Application starts with a welcome page
User navigates to the palindrome checker
User inputs text
Input is processed
Result is displayed (Palindrome / Not Palindrome)
// objective

To understand string manipulation
To learn branching and merging in Git
To build a simple and user-friendly checker application
// what are topics it will be covered

Strings and conditionals
Functions
Git branching (feature branch for welcome page)
Basic UI / input handling
// which usecase

Checking words like "madam", "racecar"
Checking sentences with spaces
Learning beginner-level project structure
Demonstrating Git workflow with multiple branche
